# 🚀 Projeto Atual

## Kellera Bio Aura System

Tecnologia criada para transformar sinais biológicos em respostas visuais e interativas.

### Integrações atuais:
- 🌡️ Sensor de temperatura DS18B20
- 🌈 LED RGB HW-479
- ⚡ ESP32 DevKit V1
