* acessibilidade avançada

---

# 🧠 Visão

O KELLERA nasceu com o objetivo de reduzir progressivamente a dependência de telas, toques e interfaces tradicionais.

A proposta é criar uma nova geração de interação humano-computador através de:

* voz
* contexto
* biometria
* sensores
* intenção humana
* assistência contextual contínua

O foco inicial está em acessibilidade para deficientes visuais, mas a visão de longo prazo envolve tecnologia humana aplicada à autonomia, mobilidade e computação natural.
